mongodb常用命令
发表于 2019-10-24| 更新于 2020-04-02|mongodb
阅读量:

mongodb

mongodb概念

sql术语概念 mongodb术语概念 说明
database database 数据库
table collection 数据库表/集合
row document 数据记录行、文档
column field 数据字段/域
index index 索引
table joins 表连接，MongoDB不支持
primary key primary key 主键，MongoDB自动将_id字段设为主键

mongodb命令

创建数据库：use db_name

查看数据库：show dbs

删除数据库：db.dropDatebase()

当前数据库：db

创建集合：db.createCollection(name,options)

查看集合：show collections

删除集合：db.collection_name.drop()

插入文档: db.collection_name.insert(document)

示例：db.col.insert({title:"mongodb插入文档"，desc:"插入操作",user:"hqs"}),插入文档时会自动创建集合

查看文档: db.collection_name.find({key1:value1,key2:value2}) 类似于sql中的and查询

加pretty()以易读方式来读取数据，如：db.col.find().pretty()

常用条件查询：

操作 格式 范例 sql类似语句
= {<key>:<value>} db.col.find("by"："菜鸟教程").pretty() where by = "菜鸟教程"
< {<key>:{$lt:<value>}} db.col.find({"likes":{$lt:50}}) where likes < 50
<= {<key>:{$lte:<value>}} db.col.find({"likes":{$lte:50}}) where likes <= 50
> {<key>:{$gt:<value>}} db.col.find({"likes":{$gt:50}}) where likes > 50
>= {<key>:{$gte:<value>}} db.col.find({"likes":{$gte:50}}) where likes >= 50
!= {<key>:{$ne:<value>}} db.col.find({"likes":{$ne:50}}) where likes != 50

db.col.find({$or:[{key1:value1},{key2,value2}]}) 类似sql中的or查询

示例：db.col.find({$or:[{"by":"菜鸟教程"},{"title":"MongoDB教程"}]})

and和or联合使用：

db.col.find({"likes":{$gt:50},$or:[{"by":"菜鸟教程"},{"title":"MongoDB教程"}]})

相当于where likes > 50 and (by = "菜鸟教程" or title = "MongoDB教程")

模糊查询：

db.col.find({title:/教/})查询包含”教”字的文档

db.col.find({title:/^教/})查询以”教”字开头的文档

db.col.find({title:/教$/})查询以”教”字结尾的文档

查看所有文档：db.collection_name.find()

更新文档：db.collection_name.update(<query>,<update>,{upsert:<boolean>,multi:<boolean>,writeConcern:<document>})

query:查询条件，类似sql update后面的where

update:update的对象和一些更新的操作符($,$inc..),类似sql update后面的set

upsert:可选，如果不存在update的记录，是否插入，true为插入，默认false不插入

multi:可选，默认false只更新找到的第一条记录，若为true则更新所有记录

writeConcern:可选，抛出异常的级别

示例：db.col.update({'title','MongoDB教程'}，{$set:{'title':'MongoDB'}},{multi:true})

db.col.update({"count":{$gt:1}},{$set:{"test2":"OK"}}) 只更新第一条记录

db.col.update({"count":{$gt:3}},{$set:{"test2":"OK"}},false,true) 全部更新

db.col.update({"count":{$gt:4}},{$set:{"test2","OK"}},true,false) 只添加第一条

db.col.update( { "count" : { $gt : 5 } } , { $set : { "test5" : "OK"} },true,true ); 全部添加

db.col.update( { "count" : { $gt : 15 } } , { $inc : { "count" : 1} },false,true ); 全部更新

db.col.update( { "count" : { $gt : 10 } } , { $inc : { "count" : 1} },false,false );只更新一条记录

save()方法：db.collection.save(<document>,{writeConcern:<document>})通过传入的文档来替换已有的文档

删除文档: db.collection.remove(<query>,<justOne>)

query:删除文档的条件

justOne:设置为true或1则删除一个文档，默认false,删除所有匹配文档。

示例：db.col.remove({'title':'MongoDB教程'})

目前remove方法已经过时了,remove方法不会真正释放空间，需要执行db.repairDatabase()回收磁盘空间

官方推荐db.col.deleteOne({'title','MongoDB教程'})和db.col.deleteMany({}'title','MongoDB教程'})

删除所有文档: db.col.remove({}) 类似sql的truncate命令

Limit()方法: db.collection_name.find().limit(NUMBER)

Skip()方法: db.collection_name.find().skip(NUMBER) 跳过指定数量的数据

Sort()方法: db.collection_name.find().sort({KEY:1}) KEY为指定排序的字段，1为升序排列，-1为降序排列

聚合: db.collection_name.aggregate(AGGREGATE_OPERATION)

示例：db.col.aggregate([{$group : {_id : "$by_user" , num_tutorial : {$sum : 1}}}])

相当于 select by_user,count(*) from col group by by_user

常用聚合表达式：

$sum,$avg,$max,$min,etc
